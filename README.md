### Evallo_Assignment

Evallo_Assignment is a front-end implementation using React (via Vite) to demonstrate UI, component architecture, routing, and state management.

## Features  
- Built with React + Vite for fast development and hot module replacement.  
- Clean component-based architecture and responsive design.  
- ESLint configured to enforce coding standards.  
- Easily extendable to integrate backend APIs or additional features.

### Tech Stack

- **Frontend**: React  

- **Bundler / Dev Tool**: Vite  

- **Linting**: ESLint  

- **Languages**: JavaScript, HTML, CSS  

- **Package Manager**: npm (as per `package.json`)

### Getting Started

## Prerequisites  
- Node.js (v14+ recommended)  

- npm (v6+ or higher)  

### Installation  
 
## Clone the repository:  

git clone https://github.com/ibrahim9492/Evallo_Assignment.git

## Navigate into the project folder:

cd Evallo_Assignment

## Install dependencies:

npm install

## Running Locally

To start the development server with hot-reload:

npm run dev

### Usage

Once the app is running:

Navigate through various pages via the UI (e.g., Home, About, Dashboard).

Explore how components are structured and how state is managed.

Extend the app by adding new routes, components, or backend integration.
